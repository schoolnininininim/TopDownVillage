## Mind Roulette Duo

간단한 모바일 2인용 픽셀 아트 탑뷰 심리전 게임입니다.

게임 요약
- 제목: Mind Roulette Duo
- 플랫폼: 모바일 기기 1대에서 2인 플레이
- 장르: 심리전 · 운빨 · 턴제 선택지

게임 규칙 요약
- 각 라운드마다 10칸의 챔버에 1~3개의 탄알을 무작위로 장전합니다.
- 플레이어는 자기 차례에 `Self-shot` 또는 `Opponent-shot`을 선택합니다.
- 규칙:
  - 상대에게 쏘고 탄환이 안 나오면 → 턴이 상대에게 넘어감.
  - 상대에게 쏘고 탄환이 나오면 → 상대 1점 감점, 탄환 소모, 턴은 상대에게 넘어감.
  - 자기에게 쏘고 탄환이 나오면 → 본인 1점 감점, 탄환 소모, 턴은 상대에게 넘어감.
  - 자기에게 쏘고 탄환이 안 나오면 → 내 턴 유지.
- 장전된 모든 총알이 소모되면 라운드 종료. 총 라운드 수는 10라운드이며, 페널티(감점)가 적은 플레이어가 승리합니다.

실행 방법
1. 이 저장소를 로컬에 클론합니다.
2. `index.html`을 모바일 브라우저(또는 데스크탑에서 모바일 뷰)로 열어 플레이합니다.

참고
- 구현은 단일 파일(`index.html`)로 되어 있으며, 캔버스 기반 픽셀 아트, 총구 플래시와 탄환 이동 애니메이션을 포함합니다.
- 개선하거나 추가하고 싶은 기능(사운드, 승리 화면, 애니메이션 고도화 등)이 있으면 요청해주세요.

Enjoy!
